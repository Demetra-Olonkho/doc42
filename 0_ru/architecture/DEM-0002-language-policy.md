# DEM-0002 — Языковая политика проекта

Версия: 0.1 (Draft)

Статус: Accepted

---

# 1. Назначение

Настоящий документ определяет языковую политику проекта Demetra Olonkho.

Политика распространяется на:

* модель знаний;
* проектную документацию;
* процессы разработки;
* терминологию проекта.

---

# 2. Основной принцип

Все языки являются равноправными участниками модели знаний.

Ни один язык не обладает привилегированным положением внутри модели знаний, форматов данных или API.

---

# 3. Рабочие языки проекта

Для разработки проекта используются рабочие языки.

Первый рабочий язык:

```
Русский
```

Второй рабочий язык:

```
Английский
```

Рабочие языки используются для:

* обсуждения архитектуры;
* подготовки проектной документации;
* оформления архитектурных решений;
* сопровождения проекта.

Статус рабочего языка не предоставляет языку особых прав внутри модели знаний.

---

# 4. Принцип круглого стола

Проект рассматривает все языки как равноправные.

Выбор рабочих языков определяется исключительно практическими соображениями и историей возникновения проекта.

Рабочие языки не являются главными или привилегированными языками системы.

---

# 5. Терминология

Архитектурные и концептуальные термины первоначально формируются на русском языке.

После согласования термина определяется его официальный английский эквивалент.

Обе формы фиксируются в глоссарии проекта.

---

# 6. Международная поддержка

Архитектура проекта должна изначально проектироваться как многоязычная.

Поддержка дополнительных языков не должна требовать изменения базовой модели знаний.

---

# 7. Статус

Данный документ считается одним из основополагающих документов проекта.

Конец документа.
